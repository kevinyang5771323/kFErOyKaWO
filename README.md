# 酒店管理系统 java103

## 项目概述

本项目是基于SpringBoot和Vue的前后端分离的酒店管理系统。系统支持酒店预订、入住管理等多种功能，并提供可扩展的协同过滤推荐算法、支付宝沙箱环境以及AI模型对话等增强选项。系统设计灵活，可根据客户需求定制添加功能。

## 技术栈

- **系统环境**：Windows，Mac
- **开发环境**：IDEA、JDK 1.8、Mysql
- **技术栈**：
  - 后端：Java、SpringBoot
  - 前端：Vue
  - 数据库：Mysql

## 功能模块

### 用户前台

- 用户登录
- 用户注册
- 浏览酒店信息
- 预定酒店
- 查看订单

### 后台管理

#### 总管理员

- 首页统计图

#### 客房管理

- 房间类型
- 房间信息
- 房间物品
- 预定方式

#### 入住管理

- 入住登记
- 退房登记
- 订单信息管理
- 客户信息管理

#### 系统管理

- 操作员管理
- 服务员管理部门信息管理
- 角色管理

#### 操作管理员

- 首页统计图
- 入住管理
  - 入住登记
  - 退房登记
  - 订单信息管理
  - 客户信息管理

## 系统角色

- 用户前台：普通用户
- 后台管理：
  - 总管理员
  - 客房管理员
  - 入住管理员
  - 系统管理员
  - 操作管理员

## 运行环境

- 后端：Java 1.8及以上环境
- 前端：现代浏览器环境
- 数据库：Mysql兼容数据库
